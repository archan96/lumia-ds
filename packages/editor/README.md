# @lumia/editor

The core editor package for Lumia DS.

## Installation

```bash
pnpm add @lumia/editor
```

## Usage

```tsx
import { LumiaEditor } from '@lumia/editor';

function App() {
  return <LumiaEditor placeholder="Start typing..." />;
}
```

## Development

To view the editor components in Storybook:

```bash
pnpm storybook
```
